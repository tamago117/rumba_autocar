# rumba_autocar
navigation practice using roomba

## Installation
```
cd catkin_ws/src
git clone https://github.com/tamago117/rumba_autocar.git
cd ..
catkin_make
```

## simple demo
```
roslaunch rumba_autocar wpTrack.launch
```
